Język kwerend

Dla dziedziny akcji D możeny definiować zapytania zwane kwerendami.

Syntaktyka

We wszystkich kwerendach podanie agenta wykonujacego daną akcję nie jest wymagane. Jeżeli agent nie zostanie podany, akcja jest wykonywana przez któregokolwiek agenta.

necessary executable ω1 A1, ..., ωn An

Program działań ((A1, ω1), ..., (An, ωn)) jest wykonywalny zawsze.

possibly executable ω1 A1, ..., ωn An

Program działań ((A1, ω1), ..., (An, ωn)) jest wykonywalny kiedykolwiek.

necessary γ after ω1 A1, ..., ωn An from π

Wykonanie programu działań ((A1, ω1), ..., (An, ωn)) z dowolnego stanu spełniającego warunek π zawsze prowadzi do stanu spełniającego warunek γ.

possibly γ after ω1 A1, ..., ωn An from π

Wykonanie programu działań ((A1, ω1), ..., (An, ωn)) z dowolnego stanu spełniającego warunek π kiedykolwiek prowadzi do stanu spełniającego warunek γ.

typically γ after ω1 A1, ..., ωn An from π

Wykonanie programu działań ((A1, ω1), ..., (An, ωn)) z dowolnego stanu spełniającego warunek π na ogół prowadzi do stanu spełniającego warunek γ.

necessary accessible γ from π

Wykonanie dowolnego programu działań z dowolnego stanu spełniającego warunek π zawsze prowadzi do stanu spełniającego warunek γ.

possibly accessible γ from π

Wykonanie dowolnego programu działań z dowolnego stanu spełniającego warunek π kiedykolwiek prowadzi do stanu spełniającego warunek γ.

typically accessible γ from π

Wykonanie dowolnego programu działań z dowolnego stanu spełniającego warunek π na ogół prowadzi do stanu spełniającego warunek γ.

necessary ω in ω1 A1, ..., ωn An

Wykonawca ω jest zawsze zaangażowany w realizację programu działań ((A1, ω1), ..., (An, ωn)).

possibly ω in ω1 A1, ..., ωn An

Wykonawca ω jest kiedykolwiek zaangażowany w realizację programu działań ((A1, ω1), ..., (An, ωn)).
