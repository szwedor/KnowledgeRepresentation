Instrukcja wartości

α after ω1 A1, ..., ωn An
gdzie n≥1

Formuła α jest zawsze prawdziwa po wykonaniu sekwencji akcji A1 przez ageneta ω1, ..., An przez ageneta ωn. Podanie agenta wykonującego akcję nie jest wymagane. Jeżeli agent nie zostanie podany, akcja wykonywana jest przez kogokolwiek.

initially α

Jeżeli dla powyższej intrukcji n = 0, to musimy użyć skróconego zapisu. Oznacza, że formuła α jest spełniona dla stanu początkowego.

observable α after ω1 A1, ..., ωn An
gdzie n≥1

Formuła α jest czasem prawdziwa po wykonaniu sekwencji A1 przez agenta ω1, ..., An przez agenta ωn. Podanie agenta wykonującego akcję nie jest wymagane. Jeżeli agent nie zostanie podany, akcja jest wykonywana przez kogokolwiek.
